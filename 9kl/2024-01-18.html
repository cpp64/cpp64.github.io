I вариант

  1. Вход: некоторая строка s.

    Выход: количество символов строки, которые являются заглавными английскими буквами.

  2. Вход: некоторая строка s.

    Выход: сначала символы, стоявшие на нечетных позициях, затем - на четных.

  3. Вход: некоторая строка s.

    Выход: кол-во гласных.

II вариант

  1. Вход: некоторая строка s.

    Выход: количество символов строки, которые являются цифрами.

  2. Вход: некоторая строка s из малых английских букв.

    Выход: количество символов, которые относятся к 1 половине английского алфавита.

  3. Вход: некоторая строка s.

    Выход: кол-во спец. символов - одного из ! ; % : ? * ( ) _ + - = / .

III вариант

  1. Вход: некоторая строка s.

    Выход: количество символов строки, которые являются малыми английскими буквами.

  2. Вход: некоторая строка s.

    Выход: строка s, разбитая на группы по 3 подряд идущих символа. Например, из 123abcdef должно получиться 123 abc def.

  3. Вход: некоторая строка s из цифр.

    Выход: кол-во каждой из цифр в строке в виде набора пар вида (цифра; кол-во).

IV вариант

  1. Вход: некоторая строка s.

    Выход: кол-во символов строки, равных '%', '~' или '@'.

  2. Вход: некоторая строка s.

    Выход: сначала символы, стоявшие на четных позициях, затем - на нечетных.

  3. Вход: некоторая строка s из малых английских букв.

    Выход: разность номеров в алфавите самой большой и самой маленькой букв в строке s. Например, для строки abde - самая большая e, самая маленькая a. Разница их номеров равна 4.

V вариант

  1. Вход: некоторая строка s.

    Выход: строка полученная со входа, в которой все символы 'a' заменены на '!'.

  2. Вход: некоторая строка s из малых английских букв.

    Выход: символы, которые относятся ко 2 половине английского алфавита.

  3. Вход: некоторая строка s.

    Выход: кол-во согласных.
