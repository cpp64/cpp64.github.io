I вариант

  1. Вход: количество чисел n и сами n чисел.

    Выход: кол-во нулей среди полученных n чисел.

  2. Вход: количество чисел n и сами n чисел.

    Выход: номер первого слева нуля среди полученных n чисел.

  3. Вход: n, m - количество строк и столбцов в двумерном массиве.

    Создайте массив указанных размеров и заполните его нулями.

II вариант

  1. Вход: количество чисел n и сами n чисел.

    Выход: сумма полученных n чисел.

  2. Вход: количество чисел n и сами n чисел.

    Выход: количество пар одинаковых соседних чисел среди полученных n чисел. Номера соседних чисел отличаются ровно на 1.

  3. Вход: n, m - количество строк и столбцов в двумерном массиве.

    Создайте массив указанных размеров и считайте все его элементы с клавиатуры. На выход выведите элемент, находящийся на пересечении 5 строки и 3 столбца.

III вариант

  1. Вход: количество чисел n и сами n чисел.

    Выход: минимум среди полученных n чисел.

  2. Вход: количество чисел n и сами n чисел.

    Выход: номер первого справа нуля среди полученных n чисел.

  3. Вход: n, m - количество строк и столбцов в двумерном массиве.

    Создайте массив указанных размеров и заполните его значением -1.

IV вариант

  1. Вход: количество чисел n и сами n чисел.

    Выход: количество отрицательных чисел среди полученных n чисел.

  2. Вход: количество чисел n и сами n чисел.

    Выход: количество пар различных соседних чисел среди полученных n чисел. Номера соседних чисел отличаются ровно на 1.

  3. Вход: n, m - количество строк и столбцов в двумерном массиве.

    На выход выведите элемент, находящийся на пересечении 1 строки и 2 столбца.
