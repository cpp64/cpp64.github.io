Задания

I вариант

  1. Вход: некоторое натуральное число.

    Выход: YES, если нет ни одной цифры 0 в числе и NO - в противном случае.

  2. Вход: некоторое натуральное число.

    Выход: YES, если в числе от двух до трех цифр 0 и NO - в противном случае.

  3. Вход: n - количество чисел в последовательности. Затем последовательность из n чисел.

    Выход: количество чисел в последовательности, которые не делятся ни на 2 ни на 3.

II вариант

  1. Вход: некоторое натуральное число.

    Выход: количество цифр 2 в этом числе.

  2. Вход: некоторое натуральное число.

  3. Вход: n - количество чисел в последовательности. Затем последовательность из n чисел.

    Выход: количество двузначных чисел в последовательности.
