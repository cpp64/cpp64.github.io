  1. Укажите количество корпусов.

    Укажите количество уроков в данные дни недели.
                            
  2. Заполните общую информацию по параллелям.
    В каждой строке описывается количество классов в соответствующей параллели, а также количество кабинетов в корпусе.
    Номера кабинетов начинаются с 1 и могут быть только числами. Букв в номерах кабинетов и корпусов не предусмотрено.
    Номер корпуса нигде дополнительно указывать не требуется.
    Скачать таблицу для параллелей
  3. Выберите файл с заполненной таблицей параллелей.
  4. Заполните таблицу распределения часов.
    Столбцы с решеткой # оставить пустыми.
    Мин - минимальное количество уроков в день.
    Макс - максимальное количество уроков в день.
    Часы - количество часов в неделю.
    Кабинеты могут быть только в диапазоне от 1 до 9999.
    Учителя, ведущие урок в одно и то же время могут объединяться по 2, по 3 и т.д. Для этого перечислить их ФИО и номера кабинетов через наклонную черту /.
    Кабинеты могут задаваться через запятую или тире, например 1,3-4 отвечает за 1,3,4 кабинеты.
    Скачать таблицу распределения часов
  5. Выберите файл с заполненной таблицей распределения часов.
  6. Составьте расписание.
